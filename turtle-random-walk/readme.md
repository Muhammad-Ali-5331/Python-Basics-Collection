# 🐢 Turtle Random Walk

A generative art script that uses Python's Turtle library to create a colorful, non-deterministic "Random Walk" visualization.

## Features
- **True RGB Randomization:** Each step features a unique color generated from over 16 million possibilities.
- **Fast Rendering:** Optimized speed and hidden turtle cursor for smooth, immediate visual feedback.
- **Geometric Pathfinding:** The turtle moves in randomized 90-degree increments, creating a complex grid-like pattern.
- **Interactive Exit:** The graphics window stays open until clicked, allowing for inspection of the final artwork.

## How to Run
1. Ensure you have Python installed (Turtle is included in the Standard Library).
2. Run the command: `python main.py`
3. Watch the generative pattern unfold on the black canvas.
