# Python Basics Collection 🐍

Welcome to the **Python Basics Collection** – a curated set of over 40 mini-projects, algorithms, and coding exercises designed to demonstrate fundamental to intermediate Python concepts. Each project is self-contained, well-documented, and serves as a practical reference for both beginners and seasoned developers looking to refresh their skills.

---

## 🔍 Overview

This repository is a **learning-oriented** collection of Python scripts and small applications. Each folder focuses on a single concept or problem, making it easy to navigate and understand. Whether you're practicing sorting algorithms, building terminal games, or exploring Python's `turtle` graphics, you'll find a relevant example here.

**Key highlights:**
- ✅ 40+ unique Python scripts
- ✅ Covers algorithms, OOP, file I/O, data structures, and more
- ✅ Clean, readable code with comments
- ✅ Many projects include ASCII art or terminal-based interactivity
- ✅ Suitable for beginners and intermediate learners

## 🚀 Getting Started

### Prerequisites
- Python 3.7 or higher
- No external libraries required (except `tkinter` which comes with Python)

### Clone the Repository
```bash
git clone https://github.com/Muhammad-Ali-5331/Python-Basics-Collection.git
cd Python-Basics-Collection
```

### ▶️ Run Any Project
```bash
cd bubble-sort
python bubble_sort.py
```

### 💡 How to Use
1. Browse the folders and pick a topic that interests you.

2. Read the inline comments and the project-specific README.md (where available).

3. Run the script and interact with it.

4. Experiment – modify the code, add features, or break it to learn.

💬 Many projects include ASCII animations, colorful terminal output, or interactive prompts – so have fun exploring!
