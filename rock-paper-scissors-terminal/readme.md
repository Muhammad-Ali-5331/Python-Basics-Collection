# 🪨📄✂️ Rock Paper Scissors

A classic CLI-based game featuring randomized computer opponents and ASCII art visuals.

## Features
- **Visual Feedback:** Displays ASCII art for each move (Rock, Paper, or Scissors).
- **Randomized AI:** Uses Python's `random` module for unpredictable computer moves.
- **Score Logic:** Full implementation of game rules including draw conditions and win/loss tracking.

## How to Play
1. Run the script: `python main.py`
2. Enter `0` for Rock, `1` for Paper, or `2` for Scissors.
3. Compare your choice against the computer to see who wins!
