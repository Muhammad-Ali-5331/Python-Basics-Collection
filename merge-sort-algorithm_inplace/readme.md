# 📉 Merge Sort Implementation

A high-performance Python implementation of the Merge Sort algorithm, showcasing recursive programming and efficient data merging techniques.

## Features
- **Divide and Conquer:** Recursively breaks down complex lists into manageable sub-problems.
- **Stable Sorting:** Maintains the relative order of equal elements during the merge phase.
- **Optimized Merging:** Uses a multi-pointer approach to synchronize the reconstruction of the sorted array.

## Performance
- **Time Complexity:** $O(n \log n)$ in best, average, and worst cases.
- **Space Complexity:** $O(n)$ due to temporary subarray creation.

## Usage
1. Open `main.py`.
2. Modify the `Array` variable in the `if __name__ == "__main__":` block.
3. Run the script to see the step-by-step sorting result in the terminal.
